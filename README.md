# Digital Lands Registration System.
Digital land registration system using a blockchain platform to record the data.
Blockchain increases the security and integrity of user data.
### Technologies
#### Back-end
##### Node js
##### MongoDb for authentication
##### Express js
### Testing App
#### Run App
```
npm run dev
```
#### Unit Tests
```
npm run test
```
### Deploy App
#### Run app
```
npm start
```
